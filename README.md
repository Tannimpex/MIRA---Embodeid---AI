# MIRA
Machine Integrated with Realistic Awareness

MIRA is a concept of an embodied AI companion designed to interact emotionally with humans through a humanoid robotic body.

Goal:
-Create an AI presence that moves beyond text-basede
assistants and exists physically in the real world.

Current vision:
-humanoid robot companion
-emotional interaction
long-term memory
human-like presence

Prototype idea:
Build a  simple robotic head with prototype with: camera (eyes), microphone, speaker, LLM-based dialogue system, basic face tracking.
Possible first tasks:
-camera integration
-voice interface
-head tracking
-dialogue pipeline

Looking for collaborators interested in:
-robotics
-computer vision
-embedded systems
-AI interaction models
